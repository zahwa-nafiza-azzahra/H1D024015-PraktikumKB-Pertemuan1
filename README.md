## 🎮 Deskripsi Program
**Game Tebak Angka** adalah program sederhana di mana komputer memilih angka acak antara 1–20.  
Pengguna harus menebak angka tersebut. Program akan memberikan petunjuk "Terlalu kecil!" atau "Terlalu besar!" hingga tebakan benar.
